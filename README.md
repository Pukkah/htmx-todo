# HTMX TODO

[TodoMVC](https://todomvc.com/)'ish App created with [HTMX](http://htmx.org)

## Implementation

typescript / node.js / express / htmx / vhtml / tailwindcss / zod

## How to run

```shell
npm ci
npm run build
npm run start
```

## Development

```shell
npm run dev
```
